**************************************English to Hindi Dictionary*****************************************

This project contains following files and folders-
--->Input : A folder to store the codebook, training utterrences and the initial lambda 
--->Output : A folder to store the final lambda for all words
--->Live_Data : A folder to store the utterrences and lambdas for live training
--->input_file.txt : stores the word spoken by the user in text format.
--->input_file.wav : audio file for the word spoken by the user.

Code files :
--->Header files(.h) :
		-> Form1.h
		-> HMM.h

--->CPP files(.cpp) :
		-> English_To_Hindi_Translator.cpp

************************************************************************************************

Instructions to run the project--------------------------------------

*** Open the solution in visual studio 2010.
*** Run the project. (keyboard shortcut - F5)
